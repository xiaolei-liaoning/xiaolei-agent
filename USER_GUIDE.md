# 用户指南

欢迎使用小雷版小龙虾AI Agent系统！本指南将帮助您快速了解和使用这个强大的AI助手。

## 🎯 您能做什么

小雷版小龙虾AI Agent是一个智能助手，可以帮助您完成各种任务：

### 📊 数据获取与分析
- **爬取热门内容**: 获取微博、百度、B站、抖音、GitHub等平台的热搜和热门内容
- **数据分析**: 自动分析数据并生成可视化报告
- **市场数据**: 获取各类市场信息和趋势

### 🤖 自动化任务
- **工作流自动化**: 创建自动化工作流，让重复性任务自动完成
- **GUI自动化**: 自动操作电脑上的应用程序
- **定时任务**: 设置定时执行的任务

### 🌐 网络功能
- **网页抓取**: 从各种网站抓取所需信息
- **搜索功能**: 智能搜索和整理信息
- **第三方服务**: 集成多种第三方API服务

### 📝 内容处理
- **翻译**: 多语言翻译功能
- **OCR识别**: 识别图片中的文字
- **内容生成**: 辅助生成各类内容

### 💡 智能助手
- **深度思考**: 复杂问题的深度分析和解答
- **多轮对话**: 支持连续的对话交流
- **记忆功能**: 记住您的偏好和历史对话

## 🚀 快速开始

### 第一步：访问系统

打开浏览器，访问以下任一地址：

- **主页**: http://localhost:8001
- **工作流编辑器**: http://localhost:8001/workflow_editor
- **Coze 聊天**: http://localhost:8001/coze

### 第二步：开始对话

在聊天界面中，直接输入您的问题或需求，例如：

```
爬取微博热搜前10条
```

系统会自动识别您的意图并执行相应操作。

### 第三步：查看结果

系统会返回执行结果，您可以直接查看或下载相关文件。

## 🎨 工作流编辑器

工作流编辑器让您可以可视化地创建自动化任务：

1. 访问 http://localhost:8001/workflow_editor
2. 拖拽组件到画布
3. 连接组件形成工作流
4. 设置参数和触发条件
5. 保存并执行工作流

### 常用工作流示例

**每日新闻汇总**:
1. 爬取各大平台热搜
2. 整理和去重
3. 生成报告
4. 发送到邮箱

**数据监控**:
1. 定时获取数据
2. 分析变化
3. 生成图表
4. 异常提醒

## 📱 使用技巧

### 1. 明确表达需求

✅ 好的表达:
```
爬取微博热搜前10条，保存为CSV文件
```

❌ 不够明确:
```
爬取微博
```

### 2. 使用具体参数

```
爬取GitHub trending pages=3 language=python
```

### 3. 多轮对话

您可以连续提问，系统会记住上下文：

```
用户: 爬取微博热搜
系统: 已获取微博热搜前10条...
用户: 再爬取百度热搜
系统: 已获取百度热搜前10条...
用户: 对比这两个平台的热搜
系统: 正在分析对比...
```

### 4. 查看历史

在历史记录中查看之前的对话和结果，方便回顾和重复使用。

## ❓ 常见问题

### Q: 系统支持哪些平台？

A: 目前支持微博、百度、B站、抖音、GitHub、知乎、今日头条、豆瓣等12+主流平台。

### Q: 爬取的数据可以导出吗？

A: 可以，系统支持CSV、Markdown等多种格式导出。

### Q: 工作流可以定时执行吗？

A: 可以，您可以在工作流中设置定时任务。

### Q: 系统会记住我的对话吗？

A: 会，系统有记忆功能，可以记住您的对话历史和偏好。

### Q: 如何查看系统状态？

A: 访问 http://localhost:8001/monitor 查看系统监控信息。

### Q: 遇到问题怎么办？

A: 您可以查看API文档 http://localhost:8001/docs 或联系技术支持。
