**影响**: 工作流编辑器页面可以正常加载

**问题**: 函数定义在局部作用域，onclick 无法访问

**修复**: 在函数定义后立即暴露到 window 对象  
**影响**: 所有按钮功能正常工作

#### 5. API 端点同步问题 ❌ → ✅
**问题**: 端口 8001 缺少 `/api/skills` 端点  
**文件**: `main.py`  
**修复**: 
- 添加 `List` 类型导入
- 复制 `get_all_skills()` 函数
- 注册 `/api/skills` 和 `/api/skills/search` 端点

**影响**: 工作流编辑器可以从端口 8001 加载技能列表

#### 6. 类型导入缺失 ❌ → ✅
**问题**: `NameError: name 'List' is not defined`
