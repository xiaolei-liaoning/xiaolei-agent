# /scrape — 数据爬取

从各平台爬取热搜/热门数据。

## 用法
```
/scrape weibo top10
/scrape zhihu hot
/scrape bilibili trending
```

## 支持平台
- weibo — 微博热搜
- zhihu — 知乎热榜
- bilibili — B站热门
- baidu — 百度热搜
- douyin — 抖音热门
- github — GitHub trending
