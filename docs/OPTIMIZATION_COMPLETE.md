### 2. 意图识别优化 ✅

#### 改进内容

**A. 扩展意图模式库**
- 从18个意图扩展到**24个意图**
- 为每个意图添加**示例句子**便于维护
- 增加了音乐、音量、截图、文件等专用意图

**B. 引入同义词模糊匹配**
```python
self.synonyms = {
    "打开": ["开启", "启动", "运行", "开"],
    "关闭": ["关掉", "停止", "退出", "关"],
    "搜索": ["查找", "查询", "检索", "找", "搜"],
    "聊天": ["说说话", "聊聊", "随便聊", "聊聊天"],
    "播放": ["放", "播放", "听"],
}
```

**C. 优化置信度计算算法**
新的置信度计算考虑5个维度：
1. **模式匹配密度** (40%)
2. **关键词位置** (30%) - 越靠前权重越大
3. **同义词匹配** (20%)
4. **上下文增强** (10%)
5. **意图权重** (×)

**D. 增加澄清功能**
- 低置信度（<0.3）会标记需要澄清
- 返回 `needs_clarification: True/False`

## 关键改进点

### 1. 智能意图匹配
- 支持同义词识别，提高用户体验
- 位置敏感算法，更符合自然语言习惯

## 使用建议

### 对于意图识别
- 如果发现未识别的输入，添加到同义词库或模式库
- 调整意图权重可以影响匹配优先级
- 利用 `needs_clarification` 字段优化对话流程

### 对于系统集成
- 使用 `core/task_interfaces.py` 定义的接口避免循环依赖
- 在生产环境中启用 DEBUG 级别日志调试
- 可以选择性禁用BFS/RAG功能以提高速度
