## 🐛 常见问题排查

### 问题1: 下拉框不显示

**可能原因**: 
- CSS未加载
- HTML结构错误

**解决方法**:
```bash
# 检查服务是否正常运行
curl http://localhost:8001/coze

# 清除浏览器缓存
Ctrl+Shift+R (Windows/Linux)
Cmd+Shift+R (Mac)
```

---

### 问题2: 拖拽无反应

**可能原因**:
- JavaScript未加载
- 事件监听器未初始化

**解决方法**:
```javascript
// 在浏览器控制台执行
console.log(typeof initDragAndDrop);
// 应该输出: "function"

// 手动初始化
initDragAndDrop();
```

---

### 问题3: 文件上传后不显示标签

**可能原因**:
- uploadedFiles数组未正确更新
- updateUploadedFilesUI函数出错

**解决方法**:
```javascript
// 在浏览器控制台检查
console.log(uploadedFiles);
// 应该显示文件数组

console.log(document.getElementById('uploaded-files'));
// 应该显示DOM元素
```
