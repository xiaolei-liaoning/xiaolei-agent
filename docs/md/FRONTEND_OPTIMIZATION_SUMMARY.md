## 🎨 设计系统

### 颜色方案
```css
/* 主色调 */
--blue-primary: #3b82f6;
--purple-primary: #8b5cf6;
--green-primary: #10b981;
--pink-primary: #ec4899;

/* 渐变色 */
--gradient-blue-purple: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
--gradient-multi: linear-gradient(-45deg, #ee7752, #e73c7e, #23a6d5, #23d5ab);
```

### 圆角规范
- 小圆角: 8px (按钮、标签)
- 中圆角: 12-16px (输入框、小卡片)
- 大圆角: 20-24px (主卡片、对话框)

### 阴影层级
- 轻阴影: `shadow-md` (常规卡片)
- 中阴影: `shadow-lg` (悬停状态)
- 重阴影: `shadow-xl` (主要功能区)

### 动画时长
- 快速: 0.2s (按钮、链接)
- 标准: 0.3s (卡片、消息)
- 慢速: 0.5s+ (页面过渡)
