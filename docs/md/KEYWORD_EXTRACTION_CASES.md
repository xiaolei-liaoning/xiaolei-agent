## 案例1: 旅行规划助手

### 用户输入
```
我打算下个月去日本旅游，大概待7天左右。主要想去东京、大阪和京都这三个城市。
你能帮我查一下这三个地方的天气情况吗？特别是樱花盛开的季节是什么时候。
另外，我需要了解一下从北京到东京的机票价格，最好是经济舱。还有日本的签证办理流程，
需要准备哪些材料，大概多久能办下来。如果可以的话，顺便推荐一些当地的特色美食和必去景点。
```

### 系统处理

**🔍 关键词提取结果:**

```json
  "action_words": ["查询", "了解", "推荐"],
  "target_words": ["天气", "机票", "签证", "美食", "景点"],
  "entities": {
    "locations": ["日本", "东京", "大阪", "京都", "北京"],
    "times": ["下个月", "7天"],
    "numbers": ["7"]
  }
}
```

**⚙️ 生成参数:**
```json

```

## 💡 使用技巧

### 1. 表达越详细，提取越准确

✅ **好**: "查询北京明天上午10点的天气，包括温度、湿度、风力"  
❌ **差**: "天气怎么样"

### 2. 明确动作和目标

✅ **好**: "发送邮件给张三，主题是工作报告"  
❌ **差**: "给张三发个东西"

### 3. 提供关键实体信息

✅ **好**: "明天下午3点在国贸大厦开会"  
❌ **差**: "有个会"

### 4. 结构化多步骤任务

✅ **好**: "首先...然后...最后..."  
❌ **差**: 杂乱无章的描述
