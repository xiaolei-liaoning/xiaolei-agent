核心差异对比
维度 V1 角色分工型 V2 智能协作型
Agent 定义给模块起个Agent名字独立心智、目标、记忆、决策、工具
协作模式简单函数调用 / 调度器单向分发流水线 / 主从 / 评审 / 拍卖 / 混合，自动选择最优
上下文管理各模块各自维护本地状态全局统一上下文中心（GlobalContextCenter）
任务分配固定规则分发（按类型映射到角色）智能匹配 + 能力评分 + 负载均衡 + 动态调整
通信方式调度器 → Agent 单向Agent ↔ Agent 双向消息总线（AgentMessage）
故障处理无 / 单Agent故障不影响其他自动重试 + 降级 + 熔断器（CircuitBreaker）
状态同步无统一状态广播与订阅
可观测性无 / 基础日志全链路追踪 + 日志 + 审计
生命周期静态创建，运行后不管理注册 → 发现 → 执行 → 注销，完整生命周期
灵活性角色固定不可变，新增需修改调度器动态策略切换 + 热加载新Agent类型
执行模式每个Agent内部并发分身think → act → reflect 三阶段循环
