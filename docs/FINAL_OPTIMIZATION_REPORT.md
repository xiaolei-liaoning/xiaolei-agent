### 新增功能

#### 1. 消息历史记录系统 ⭐
**文件**: `static/js/coze.js`, `templates/coze.html`

**功能特性**:
- ✅ 自动保存所有消息到 localStorage
- ✅ 恢复历史聊天记录
- ✅ 导出为 JSON 格式
- ✅ 导出为 TXT 格式
- ✅ 可配置最大保存数量（默认 100 条）
- ✅ 消息搜索功能

**UI 改进**:
- 新增"恢复历史"按钮
- 新增"导出JSON"按钮
- 新增"导出TXT"按钮
- 优化"清空聊天"按钮（红色警示）

**使用示例**:
```javascript
// 自动保存（无需手动操作）
MessageHistory.add('user', '你好');
MessageHistory.add('assistant', '你好！有什么可以帮助你的？');

// 恢复历史
MessageHistory.restoreToUI();

// 导出
MessageHistory.exportAsJSON();
MessageHistory.exportAsTXT();

// 搜索
const results = MessageHistory.search('天气');
```
