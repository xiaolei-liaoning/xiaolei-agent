## 🎯 优化建议(基于真实数据)

### P0优先级: weather意图优化

**问题**: weather平均置信度仅0.56,27条低置信度记录

**可能原因**:
1. 短句识别不足("天"、"气"单独出现时匹配弱)
2. 口语化表达未覆盖("天怎么样")

**建议方案**:
```python
# 在intent_recognizer.py中扩展weather模式
"weather": IntentPattern(
    "weather", 
    ["天气", "气温", "温度", "预报", "今天天气", "明天天气", 
     "天气预报", "温度多少", "天气怎么样", "天", "气"],  # 添加单字
    weight=2.0 → 2.3  # 提高权重
)
```

**预期收益**: +5-8% weather意图准确率

---

### P1优先级: translate意图优化

**问题**: 12条低置信度记录

**可能原因**: 缺少短词("译"、"翻")

**建议方案**:
```python
"translate": IntentPattern(
    "translate", 
    ["翻译", "translate", "译成", "翻译成", "翻译一下", 
     "翻译翻译", "译", "翻"],  # 添加单字
    weight=2.0 → 2.3
)
```

---

### P2优先级: 调整chat置信度阈值

**问题**: chat意图占31.5%(0.3-0.4区间),作为兜底是正常的

**建议**: 保持现状,这是预期行为
