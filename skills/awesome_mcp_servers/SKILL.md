# Awesome MCP Servers Skill

## 📦 简介

Awesome MCP Servers Skill 让你能够访问并使用 awesome-mcp-servers 列表中的 **800+ MCP 服务器**。

## 🔧 功能

### 1. 搜索 MCP 服务器
```
搜索: "数据库"
搜索: "playwright"
搜索: "浏览器自动化"
```

### 2. 查看热门服务器
```
/mcp popular
列出最受欢迎的 MCP 服务器
```

### 3. 按分类浏览
支持的分类：
- 📂 Databases（数据库）
- 🧬 Browser Automation（浏览器自动化）
- ☁️ Cloud Platforms（云平台）
- 👨‍💻 Code Execution（代码执行）
- 🤖 Coding Agents（编码 Agent）
- 🗄️ Knowledge & Memory（知识与记忆）
- 🔒 Security（安全）
- 🌐 Social Media（社交媒体）

### 4. 快速连接
支持的快速连接服务器：
- `chroma` - 向量数据库
- `playwright` - 浏览器自动化
- `e2b` - 代码执行沙箱
- `sqlite` - SQLite 数据库
- `postgres` - PostgreSQL 数据库
- `filesystem` - 文件系统访问
- `github` - GitHub API
- `slack` - Slack 集成

### 5. 自定义连接
通过 GitHub URL 连接任意 MCP 服务器

## 🚀 使用方法

### 命令示例

#### 搜索服务器
```bash
/mcp search 数据库
/mcp search playwright
/mcp search 浏览器
```

#### 快速连接
```bash
/mcp connect chroma
/mcp connect playwright
/mcp connect e2b
```

#### 查看热门
```bash
/mcp popular
```

#### 查看分类
```bash
/mcp categories
/mcp by_category Databases
```

#### 查看已连接
```bash
/mcp list
```
